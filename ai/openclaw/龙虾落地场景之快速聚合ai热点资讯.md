# 背景
- 在龙虾刚出来的时候，我的一个诉求就是爬取某些网站的内容，然后聚合后定时推给我
- 但是当时是想用龙虾操作浏览器的方式去实现，这个思路当时实现较为复杂挡住了
- 随着公司基建的提升，最近又接着折腾了这个话题

# 结果
- 每天指定时间帮我聚合好AI热点资讯后，通过企业微信推送给我

# 实现思路
- 一个爬取数据的工具，可以选择开源工具（更灵活，但是要配置的内容也更多），也可以选择现有的服务（帮你做好这个事情，直接给你数据）
- 我这里选择现有的服务，[服务链接]：(https://aihot.virxact.com/)
- 好处：简单，不用twitter的账号、翻墙、查找相关的账号等
- 这个服务同时提供了一个skill，直接把skill安装到龙虾就可以.
- 使用这个skill的时候，配置一下ua环境变量即可
- 然后给龙虾下达指令：帮我在每天早上8:00拉取拉一下最新的精选

# 思考
- 其实这个东西，在龙虾出来，很多人已经做了，就是通过爬取工具之后，通过机器人将数据传递给需要的人
- 那我为什么要做呢，因为简单，只需要加载skill，下达指令就搞定了.
