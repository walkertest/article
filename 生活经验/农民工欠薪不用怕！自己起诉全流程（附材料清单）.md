爸爸打工被拖欠5000+工钱两年，没欠条没证据，我们自己走法院把钱要回来了，全程整理成6大关键步骤，姐妹们收藏！

# 1️⃣ 选对途径：劳动调解 vs 法院起诉
1. 网上说的最多的就是劳动调解途径。找了很多劳动调解部门，从区的到街道的，最后结论就是不受理，建议走法律渠道。其实从被告后续的一直拖延来看，普通的调解应该也不一定会生效。
2. 走法院起诉流程。

# 2️⃣ 选法院+定案件类型
- 一般来说，由被告的经常居住地或者住所地或者合同履行地人民法院管辖。这里选择了一个较近的（原告和被告都比较方便），写一份案件法院管辖的依据和说明。
- 案件类型选择劳动合同纠纷，填错会审核不通过。
- 审核挺久的，可能得1-2个星期
- 委托授权说明书需列明代理权限，权限就是特别代理权限

# 3️⃣ 最难一关：怎么拿到对方身份证信息
- 身份证信息是必须要的，没有会审核不过。
- 说一下几个途径：1. 通过关系找到他们村的人，找到身份证信息；2. 派出所（不受理）；3. 法院调查取证（这个好像要专门的律师才能做）；4. 通过第三方渠道协助核实。
- 我这里1/2/3几个方法都试了，最后通过方法4如愿获得了被告的身份信息，具体方法放评论区。

# 4️⃣ 怎么证明亲属关系？
- 可靠方法：找爸爸出生的所在地的一些机构开具证明，比如是湖北的，就在湖北公安政务服务中心微信小程序线上就可以开具，填好信息后，派出所就会打电话通知你去领（不能拍照传递），我这里委托了亲戚去领取。每个省份政务渠道不太一样，评论区告诉我你是哪个省，我帮你查。
- 还尝试了其他方法：找了村委会的，他们是无权受理的，好像有些地方又可以，可以试一下。

# 5️⃣ 准备证据
- 证明工作关系存在：聊天记录。
- 证明欠薪的金额：电话录音、聊天记录、工天记录、欠条、工友证明。
- 多个材料上最好按上手印。

# 6️⃣ 开庭到执行
- 立案可能非常慢：留的联系电话不一定打得通；可以搜法院的联系电话，持续询问，说排队时间过久，他们会适当优先处理。
- 立案前会有调解，再到法院开庭：在指定开庭前到达，等待庭长呼唤即可，可能会有延时；小金额的，庭长会上来调解，我们走得比较顺，很快就走完调解流程，大概20多分钟。说实话去之前很紧张，结果比想象中简单很多。
- 会出一个调解书，商定还款时间，如果仍然拖欠不执行，就会强制执行，具有法律效力。
- 是否需要去现场开庭：法院助理会提前电话联系，如果觉得你证据不充分，会建议去现场开庭，提升沟通效率；也可以选择线上开庭。

# 💰 最后：花了多少钱
- 打印一些材料。
- 法院开庭的12.5元费用（原告被告都可以承担）。
- 如果开庭前协调解决好的话，这个协调可能会收取一定的费用，比如200元。
